🤗 hf-mount Guide

Carga modelos de HuggingFace sin descargarlos

✨¿Qué es hf-mount?

hf-mount monta repositorios de HuggingFace como sistemas de archivos locales. Los archivos se leen bajo demanda — no se descarga todo el modelo, solo lo que necesitas. Funciona via FUSE o NFS, con carga lazy de archivos.

🚀Guía de Inicio Rápido

1

Instala hf-mount

curl -fsSL https://raw.githubusercontent.com/huggingface/hf-mount/main/install.sh | sh

2

Monte un modelo público

hf-mount start repo openai-community/gpt2 /tmp/gpt2

(Usa --hf-token $HF_TOKEN para modelos privados)

3

Usa con Python/Transformers

from transformers import AutoModelForCausalLM

# Carga desde path montado - NO descarga
model = AutoModelForCausalLM.from_pretrained("/tmp/gpt2")
4

Ejecuta el servidor API (opcional)

pip install flask transformers torch
python api_server.py

Luego abre http://localhost:5000
